# 一、项目背景

|          |                                                                                                                                                                                              |
| -------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **业务背景** | 外卖业务非常依赖线下场景化信息，尤其对天气有着非常高的敏感度。比如下雨、下雪、高温、低温、冰冻等异常天气可以极大地提升用户点外卖的意愿；天气顺应节气有四时变化，例如惊蛰时节常有雷雨天气，标志着春雷的开始；小雪节气则预示着降雪的可能性增加。我们期望强化站内 24 节气相关表现，需要从设计、供给等多个方面进行策略升级，视觉层面也需提升用户对节气大促的认知和记忆，增强活动影响力。 |
| **设计目标** | 提升细分场域的售卖增长                                                                                                                                                                                  |
# 二、设计策略和亮点
|          |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| -------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **设计策略** | 1. **在统一的视觉框架下演绎动态化的季节演变：**以节气罗盘、袋鼠等元素构建系列活动的仪式感，结合动植物塑造可探索的微观世界，传递自然哲学；通过人格化袋鼠的差异化设计，将品牌转化为可信赖的四季伙伴，让传统文化符号沉淀为用户个人的情感记忆。<br>    <br>2. **强化功能性与叙事性的融合：**将 “应季而食” 理念转化为具体的推荐菜品与文化卡片，为用户提供兼具养生价值与决策便利的实用功能；同时，借助罗盘等隐喻性元素清晰锚定信息，将品牌资产自然融入场景，保障体验的流畅度与直觉感受。                                                                                                                                                                                                                                                                                 |
| **设计亮点** | 1. **专业沉淀：① 情感化设计思路：**运用动植物等自然符号与人格化的袋鼠IP，将抽象节气转化为可感知、可共情的连续性生命故事。不仅唤醒了用户的文化记忆与情感共鸣，更将品牌升华为一位懂生活、有温度的长期陪伴者； ② **通过AI驱动+人脑决策，形成复杂创意的批量化生产的3步设计法**：**a. 前期-无线脑暴与概念可视化**：通过AI进行高速、大规模、低成本的发散创意，罗列各节气的代表元素，建立风格、构图、和元素组合的方案，形成故事脚本库；**b. 中期-方案推衍与元素精准生成**：通过筛选锚定最优方案，并进入“精细化提示词”阶段，通过反复“投喂”参考图和调整关键词，精准控制AIGC的输出，逐步将所有创意点整合、优化，推衍出最终方案的雏形；**c. 后期-风格统一与批量生成**：通过固定一组“种子值”和核心提示词模板，AIGC能够保证为24个节气所生成的素材（如罗盘、袋鼠、背景）在画风、色调和质感上保持高度统一，形成系列感<br>    <br>2. **业务价值**：系列化的节气情感化设计给业务带来正向收益，用户节气心智明显加强。首页浮标平均点击率提升20.7%、会场曝光UV最高提升165%，访购率平均提升18.6%。 |
