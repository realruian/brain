## 1. 背景与动机

### 1.1 问题场景

校招季面试官通常需要同时准备多位候选人的面试，每位候选人的背景差异很大：有人论文丰富，有人实习经验多，有人竞赛突出。面试官需要：

1. **了解候选人背景**：阅读简历 → 搜索论文（arXiv/Google Scholar）→ 查看GitHub仓库 → 找到个人主页 → 核实专利
2. **设计针对性问题**：根据候选人的研究方向、实习经历、技术栈，设计"问到点上"的问题
3. **准备参考答案**：写出每道问题的期望回答方向，便于现场评估
4. **输出可打印文档**：排版整齐的Word文档，面试时可打印翻阅

### 1.2 痛点

| 痛点 | 说明 | 耗时 |
| --- | --- | --- |
| 逐人搜索低效 | 每位候选人要搜论文、GitHub、专利，重复劳动 | 15-30分钟/人 |
| 问题缺乏针对性 | 忙不过来时只能用通用题库，难以深挖候选人亮点 | — |
| 格式参差不齐 | 手写笔记/文本文件，面试现场不方便翻阅 | — |
| 信息碎片化 | arXiv、GitHub、知乎、专利局散落在不同网站 | — |

### 1.3 目标

实现「简历 → 背景调研 → 面试问题 → 打印级文档」的全流程自动化：

由DrawIO流程图提供 · 问题反馈

单候选人全流程耗时从 **45分钟缩短至 3分钟**。

## 3. 第一步：背景自动调研

### 3.1 信息提取策略

### 3.2 互联网搜索补充


搜索策略：

1. **论文搜索**：`"论文标题" arxiv` → 获取arXiv链接、引用数、合作者

3. **专利搜索**：`"专利号 patent"` → 获取专利摘要、申请人、权利要求
4. **综合搜索**：`"候选人名 学校名"` → 获取导师信息、实验室主页、获奖记录

## 4. 第二步：个性化问题生成

### 4.1 问题设计原则

| 原则 | 说明 |
| --- | --- |
| 简历驱动 | 每道问题都要链接到简历中的具体经历 |
| 深度递进 | 先问基础概念，再追问实现细节，最后考工程应用 |
| 岗位对标 | 问题要和目标岗位的核心要求匹配 |
| 答案可评估 | 提供参考答案要点，便于面试官现场评分 |

### 4.2 问题类别模板

```bash

python3 scripts/generate_questions.py \
  --research-file /tmp/research.json \
  --resume-file /tmp/resume.txt \
  --job-desc "大模型算法工程师，侧重后训练（SFT/RLHF/DPO）" \
  --output /tmp/questions.md
```

生成的问题按以下类别组织：

**一、技术深度（基于论文/研究）** — 权重 25%

- 针对一作论文的方法论深挖
- 对比相关工作的技术选型思考
- 实验设计和结果分析能力

**二、工程能力（基于实习/项目）** — 权重 25%

- 实习中的技术方案设计
- 数据处理和模型训练的工程细节
- 问题排查和优化经验

**三、基础知识（基于岗位要求）** — 权重 20%

- 岗位核心技术的基础概念
- 算法原理和数学推导
- 框架和工具的使用经验

**四、业务理解（将经验迁移到目标业务）** — 权重 15%

- 如何将研究经验应用到实际业务
- 对目标业务场景的理解
- 技术方案的工程化思考

**五、综合素质** — 权重 15%

- 动机和职业规划
- 团队协作和沟通能力
- 学习能力和适应力

### 完整脚本源码

**research_candidate.py**

```python

#!/usr/bin/env python3
"""
候选人背景调研脚本 - 从简历文本中提取关键信息并通过搜索引擎进行背景调研。

用法:
    python3 research_candidate.py --resume-file /path/to/resume.txt [--output /path/to/research.json]

依赖: 仅使用 Python 标准库 (urllib, json, re, argparse)

Changelog:
    v1.0 (2026-03-25): 初始版本 - 简历解析 + Serper 搜索 + 结构化 JSON 输出
"""

import argparse
import json
import os
import re
import sys
import urllib.request
import urllib.error
import urllib.parse
from typing import Any

# ============================================================
# 简历文本解析
# ============================================================

def extract_name(text: str) -> str:
    """从简历文本头部提取候选人姓名（中文2-4字 或 英文名）。"""
    lines = [l.strip() for l in text.strip().split('\n') if l.strip()]
    if not lines:
        return ""
    # 中文姓名: 2-4个汉字独占一行（简历开头常见格式）
    for line in lines[:5]:
        m = re.match(r'^([\u4e00-\u9fff]{2,4})\s*$', line)
        if m:
            return m.group(1)
    # 英文姓名: 首行只有英文单词（2-3 个）
    for line in lines[:3]:
        m = re.match(r'^([A-Z][a-z]+(?:\s+[A-Z][a-z]+){1,2})\s*$', line)
        if m:
            return m.group(1)
    return ""

def extract_email(text: str) -> str:
    """提取邮箱地址。"""
    m = re.search(r'[\w.+-]+@[\w-]+\.[\w.-]+', text)
    return m.group(0) if m else ""

def extract_phone(text: str) -> str:
    """提取手机号（中国大陆11位 或 国际格式）。"""
    m = re.search(r'(?:\+?\d{1,3}[-\s]?)?1[3-9]\d{9}', text)
    return m.group(0) if m else ""

def extract_university(text: str) -> str:
    """提取学校名称。"""
    # 中文大学
    m = re.search(r'([\u4e00-\u9fff]{2,10}(?:大学|学院|研究院|Institute|University))', text)
    if m:
        return m.group(1)
    # 英文大学
    m = re.search(r'((?:University|Institute|College)\s+of\s+[\w\s]+)', text, re.IGNORECASE)
    if m:
        return m.group(1).strip()
    m = re.search(r'([\w\s]+(?:University|Institute|College))', text, re.IGNORECASE)
    if m:
        return m.group(1).strip()
    return ""

def extract_degree(text: str) -> str:
    """提取最高学历。"""
    for pattern in [r'(博士|Ph\.?D)', r'(硕士|Master)', r'(学士|Bachelor)']:
        m = re.search(pattern, text, re.IGNORECASE)
        if m:
            return m.group(1)
    return ""

def extract_research_areas(text: str) -> list:
    """提取研究方向关键词。"""
    areas = []
    # 研究方向/研究领域 后面的内容
    m = re.search(r'(?:研究方向|研究领域|Research\s*(?:Area|Interest)s?)[：:\s]*(.*?)(?:\n|$)', text, re.IGNORECASE)
    if m:
        raw = m.group(1)
        # 按逗号、顿号、分号分割
        areas = [a.strip() for a in re.split(r'[,，、;；/]', raw) if a.strip()]
    # 补充: 常见 AI/ML 关键词
    ml_keywords = [
        'NLP', 'LLM', '大语言模型', '自然语言处理', '计算机视觉', 'CV',
        '强化学习', 'Reinforcement Learning', 'RLHF', 'DPO',
        '推荐系统', '搜索排序', '知识图谱', 'Knowledge Graph',
        '语音识别', 'ASR', 'TTS', '多模态', 'Multimodal',
        'Transformer', 'Diffusion', 'RAG', 'Post-Training',
        '预训练', 'Pre-training', 'Fine-tuning', '微调',
        '机器学习', 'Machine Learning', '深度学习', 'Deep Learning',
    ]
    for kw in ml_keywords:
        if kw.lower() in text.lower() and kw not in areas:
            areas.append(kw)
    return areas[:15]  # 最多15个

def extract_paper_titles(text: str) -> list:
    """提取论文标题。"""
    papers = []
    # 匹配编号格式: [1] Title / 1. Title
    for m in re.finditer(r'(?:\[\d+\]|\d+[\.\)]\s+)(.+?)(?:\.|。|\n)', text):
        title = m.group(1).strip()
        if len(title) > 15 and not re.match(r'^(?:教育|工作|项目|技能)', title):
            papers.append(title)
    # 匹配引号包裹的论文名
    for m in re.finditer(r'[""「](.+?)[""」]', text):
        title = m.group(1).strip()
        if len(title) > 15:
            papers.append(title)
    # 去重
    seen = set()
    unique = []
    for p in papers:
        key = p.lower()[:30]
        if key not in seen:
            seen.add(key)
            unique.append(p)
    return unique[:10]

def extract_urls(text: str) -> dict:
    """提取简历中的链接（GitHub、个人主页等）。"""
    urls = {"github": "", "personal_page": "", "google_scholar": "", "other": []}
    for m in re.finditer(r'https?://[^\s,，)）\]]+', text):
        url = m.group(0).rstrip('.')
        lower = url.lower()
        if 'github.com' in lower:
            urls["github"] = url
        elif 'scholar.google' in lower:
            urls["google_scholar"] = url
        elif any(d in lower for d in ['linkedin.com', 'zhihu.com']):
            urls["other"].append(url)
        elif not urls["personal_page"]:
            urls["personal_page"] = url
    return urls

    """解析简历文本，提取结构化信息。"""

def serper_search(query: str, api_key: str, num: int = 5, lang: str = "zh-cn") -> list:
    """调用 Serper API 执行 Google 搜索，返回结果列表。"""
    url = "https://google.serper.dev/search"
    payload = json.dumps({"q": query, "num": num, "hl": lang}).encode("utf-8")
    req = urllib.request.Request(
        url,
        data=payload,
        headers={
            "X-API-KEY": api_key,
            "Content-Type": "application/json",
        },
        method="POST",
    )
    try:
        with urllib.request.urlopen(req, timeout=15) as resp:
            data = json.loads(resp.read().decode("utf-8"))
        results = []
        for item in data.get("organic", []):
            results.append({
                "title": item.get("title", ""),
                "link": item.get("link", ""),
                "snippet": item.get("snippet", ""),
            })
        return results
    except (urllib.error.URLError, urllib.error.HTTPError, json.JSONDecodeError) as e:
        print(f"[WARN] 搜索失败 (query={query[:40]}...): {e}", file=sys.stderr)
        return []

def search_papers(name: str, paper_titles: list, api_key: str) -> list:
    """搜索候选人的论文。"""
    found_papers = []
    # 按论文标题搜索
    for title in paper_titles[:5]:
        query = f'{title} site:arxiv.org OR site:scholar.google.com'
        results = serper_search(query, api_key, num=3, lang="en")
        for r in results:
            if any(d in r["link"] for d in ["arxiv.org", "scholar.google", "semanticscholar"]):
                found_papers.append({
                    "title": title,
                    "url": r["link"],
                    "snippet": r["snippet"],
                })
                break
    # 按名字搜索更多论文
    if name:
        query = f'{name} arxiv paper'
        results = serper_search(query, api_key, num=5, lang="en")
        existing_urls = {p["url"] for p in found_papers}
        for r in results:
            if "arxiv.org" in r["link"] and r["link"] not in existing_urls:
                found_papers.append({
                    "title": r["title"],
                    "url": r["link"],
                    "snippet": r["snippet"],
                })
    return found_papers

def search_github(name: str, github_url: str, api_key: str) -> dict:
    """搜索候选人的 GitHub 信息。"""
    info = {"profile_url": github_url, "repos": []}
    if github_url:
        # 从已知 URL 提取用户名
        m = re.search(r'github\.com/([^/\s?]+)', github_url)
        if m:
            username = m.group(1)
            query = f'site:github.com/{username}'
            results = serper_search(query, api_key, num=5, lang="en")
            for r in results:
                if f'github.com/{username}/' in r["link"]:
                    info["repos"].append({
                        "name": r["title"],
                        "url": r["link"],
                        "description": r["snippet"],
                    })
    elif name:
        # 用名字搜索 GitHub
        query = f'{name} site:github.com'
        results = serper_search(query, api_key, num=3, lang="en")
        for r in results:
            if "github.com" in r["link"]:
                info["profile_url"] = r["link"]
                break
    return info

def search_patents(name: str, api_key: str) -> list:
    """搜索候选人的专利信息。"""
    if not name:
        return []
    query = f'{name} 发明专利 OR patent'
    results = serper_search(query, api_key, num=5)
    patents = []
    for r in results:
        if any(kw in r["snippet"].lower() for kw in ["专利", "patent", "发明"]):
            patents.append({
                "title": r["title"],
                "url": r["link"],
                "snippet": r["snippet"],
            })
    return patents

def search_personal_page(name: str, university: str, api_key: str) -> str:
    """搜索候选人的个人主页。"""
    if not name:
        return ""
    query = f'{name}'
    if university:
        query += f' {university}'
    query += ' 个人主页 OR homepage'
    results = serper_search(query, api_key, num=3)
    for r in results:
        link = r["link"].lower()
        # 过滤社交媒体和招聘网站
        skip = ["zhipin.com", "lagou.com", "liepin.com", "51job.com"]
        if not any(s in link for s in skip):
            return r["link"]
    return ""

def research_candidate(resume_text: str, api_key: str) -> dict:
    """执行完整的候选人调研流程。"""
    print("[INFO] 开始解析简历...", file=sys.stderr)
    profile = parse_resume(resume_text)
    name = profile["name"]
    print(f"[INFO] 候选人: {name or '(未识别)'}", file=sys.stderr)
    print(f"[INFO] 学校: {profile['university'] or '(未识别)'}", file=sys.stderr)
    print(f"[INFO] 研究方向: {', '.join(profile['research_areas'][:5]) or '(未识别)'}", file=sys.stderr)

    report = {
        "candidate_profile": profile,
        "search_results": {
            "papers": [],
            "github": {"profile_url": "", "repos": []},
            "patents": [],
            "personal_page": "",
        },
    }

    if not api_key:
        print("[WARN] 未提供 Serper API Key，跳过搜索", file=sys.stderr)
        return report

    # 论文搜索
    print(f"[INFO] 搜索论文 ({len(profile['paper_titles'])} 篇标题)...", file=sys.stderr)
    report["search_results"]["papers"] = search_papers(
        name, profile["paper_titles"], api_key
    )
    print(f"[INFO] 找到 {len(report['search_results']['papers'])} 篇论文", file=sys.stderr)

    # GitHub 搜索
    print("[INFO] 搜索 GitHub...", file=sys.stderr)
    report["search_results"]["github"] = search_github(
        name, profile["urls"]["github"], api_key
    )

    # 专利搜索
    print("[INFO] 搜索专利...", file=sys.stderr)
    report["search_results"]["patents"] = search_patents(name, api_key)

    # 个人主页搜索
    if not profile["urls"]["personal_page"]:
        print("[INFO] 搜索个人主页...", file=sys.stderr)
        page = search_personal_page(name, profile["university"], api_key)
        report["search_results"]["personal_page"] = page
    else:
        report["search_results"]["personal_page"] = profile["urls"]["personal_page"]

    print("[INFO] 调研完成", file=sys.stderr)
    return report

def main():
    parser = argparse.ArgumentParser(
        description="候选人背景调研 - 从简历文本提取信息并搜索公开资料",
        epilog="示例: python3 research_candidate.py --resume-file resume.txt --output research.json",
    )
    parser.add_argument("--resume-file", required=True, help="简历纯文本文件路径")
    parser.add_argument("--output", default=None, help="输出 JSON 文件路径（默认输出到 stdout）")
    parser.add_argument(
        "--serper-key",
        default=os.environ.get("SERPER_API_KEY", ""),
        help="Serper API Key（也可通过环境变量 SERPER_API_KEY 设置）",
    )
    args = parser.parse_args()

    # 读取简历
    try:
        with open(args.resume_file, "r", encoding="utf-8") as f:
            resume_text = f.read()
    except FileNotFoundError:
        print(f"[ERROR] 文件不存在: {args.resume_file}", file=sys.stderr)
        sys.exit(1)
    except UnicodeDecodeError:
        # 尝试 GBK 编码
        with open(args.resume_file, "r", encoding="gbk") as f:
            resume_text = f.read()

    if not resume_text.strip():
        print("[ERROR] 简历文件为空", file=sys.stderr)
        sys.exit(1)

    # 执行调研
    report = research_candidate(resume_text, args.serper_key)

    # 输出结果
    output_json = json.dumps(report, ensure_ascii=False, indent=2)
    if args.output:
        with open(args.output, "w", encoding="utf-8") as f:
            f.write(output_json)
        print(f"[INFO] 调研报告已保存到: {args.output}", file=sys.stderr)
    else:
        print(output_json)

if __name__ == "__main__":
    main()
```

```python
#!/usr/bin/env python3
"""
面试问题 DOCX 格式化脚本 - 将 Markdown 面试问题转为打印友好的 Word 文档。

支持的 Markdown 元素:
  - 标题 (# H1, ## H2, ### H3)
  - 粗体 (**text**)
  - 表格 (| col1 | col2 |)
  - 无序列表 (- item)
  - 引用块 (> text)
  - 分隔线 (---)
  - 链接 [text](url)

用法:
    pip install python-docx  # 首次安装依赖
    python3 format_docx.py --input questions.md --output questions.docx

依赖: python-docx

Changelog:
    v1.0 (2026-03-25): 初始版本 - Markdown 解析 + DOCX 生成 + A4 打印优化

    "info_box_bg": "EBF5FB",                      # 信息框背景

def create_document(cn_font: str) -> Document:
    """创建文档并设置全局样式和 A4 页边距。"""
    doc = Document()

    # 设置 A4 页面和页边距（适合打印）
    section = doc.sections[0]
    section.page_width = Cm(21.0)
    section.page_height = Cm(29.7)
    section.top_margin = Cm(2.0)
    section.bottom_margin = Cm(2.0)
    section.left_margin = Cm(2.5)
    section.right_margin = Cm(2.0)

    # 设置默认字体
    style = doc.styles['Normal']
    font = style.font
    font.name = DEFAULT_EN_FONT
    font.size = Pt(10.5)
    font.color.rgb = COLORS["body"]
    # 设置中文字体
    rpr = style.element.get_or_add_rPr()
    rfonts = rpr.find(qn('w:rFonts'))
    if rfonts is None:
        rfonts = parse_xml(f'<w:rFonts {nsdecls("w")} w:eastAsia="{cn_font}"/>')
        rpr.append(rfonts)
    else:
        rfonts.set(qn('w:eastAsia'), cn_font)

    # 段落间距
    pf = style.paragraph_format
    pf.space_before = Pt(2)
    pf.space_after = Pt(4)
    pf.line_spacing = 1.3

    return doc

def set_run_font(run, cn_font: str, size: int = None, bold: bool = False,
                 color: RGBColor = None):
    """设置 run 的字体属性。"""
    run.font.name = DEFAULT_EN_FONT
    if size:
        run.font.size = Pt(size)
    if bold:
        run.bold = True
    if color:
        run.font.color.rgb = color
    # 中文字体
    r_elem = run._element
    rpr = r_elem.get_or_add_rPr()
    rfonts = rpr.find(qn('w:rFonts'))
    if rfonts is None:
        rfonts = parse_xml(f'<w:rFonts {nsdecls("w")} w:eastAsia="{cn_font}"/>')
        rpr.append(rfonts)
    else:
        rfonts.set(qn('w:eastAsia'), cn_font)

def add_shading(cell, color_hex: str):
    """给表格单元格添加背景色。"""
    shading = parse_xml(
        f'<w:shd {nsdecls("w")} w:fill="{color_hex}" w:val="clear"/>'
    )
    cell._tc.get_or_add_tcPr().append(shading)

def parse_inline(text: str) -> list:
    """解析行内格式，返回 (text, bold) 元组列表。"""
    parts = []
    # 处理粗体 **text**
    segments = re.split(r'\*\*(.+?)\*\*', text)
    for i, seg in enumerate(segments):
        if not seg:
            continue
        is_bold = (i % 2 == 1)
        # 处理链接 [text](url) -> "text (url)"
        seg = re.sub(r'\[([^\]]+)\]\(([^)]+)\)', r'\1 (\2)', seg)
        parts.append((seg, is_bold))
    return parts if parts else [("", False)]

def add_formatted_paragraph(doc, text: str, cn_font: str,
                            size: int = 10, color: RGBColor = None,
                            bold_all: bool = False, indent: float = None):
    """添加带行内格式的段落。"""
    p = doc.add_paragraph()
    if indent:
        p.paragraph_format.left_indent = Cm(indent)
    parts = parse_inline(text)
    for part_text, is_bold in parts:
        run = p.add_run(part_text)
        set_run_font(run, cn_font, size=size, bold=(bold_all or is_bold),
                     color=color or COLORS["body"])
    return p

def add_table(doc, rows: list, cn_font: str):
    """添加格式化表格。rows 是二维列表。"""
    if not rows or not rows[0]:
        return
    n_cols = len(rows[0])
    table = doc.add_table(rows=len(rows), cols=n_cols)
    table.alignment = WD_TABLE_ALIGNMENT.CENTER
    table.style = 'Table Grid'

    for i, row_data in enumerate(rows):
        row = table.rows[i]
        for j, cell_text in enumerate(row_data):
            if j >= n_cols:
                break
            cell = row.cells[j]
            cell.text = ""
            p = cell.paragraphs[0]
            run = p.add_run(cell_text.strip())
            if i == 0:
                # 表头样式
                set_run_font(run, cn_font, size=9, bold=True,
                             color=RGBColor(0xFF, 0xFF, 0xFF))
                add_shading(cell, COLORS["table_header_bg"])
            else:
                set_run_font(run, cn_font, size=9)
                if i % 2 == 0:
                    add_shading(cell, COLORS["table_alt_bg"])

    # 自动调整列宽
    doc.add_paragraph()  # 表格后加空行

def add_info_box(doc, text: str, cn_font: str):
    """添加信息框（引用块样式）。"""
    # 用单行表格模拟信息框
    table = doc.add_table(rows=1, cols=1)
    cell = table.rows[0].cells[0]
    add_shading(cell, COLORS["info_box_bg"].lstrip('#'))
    cell.text = ""
    p = cell.paragraphs[0]
    run = p.add_run(text)
    set_run_font(run, cn_font, size=9, color=COLORS["light"])
    doc.add_paragraph()

def convert_md_to_docx(md_text: str, cn_font: str) -> Document:
    """将 Markdown 文本转换为格式化的 DOCX 文档。"""
    doc = create_document(cn_font)
    lines = md_text.split('\n')
    i = 0
    table_rows = []
    in_table = False

    while i < len(lines):
        line = lines[i]
        stripped = line.strip()

        # 空行
        if not stripped:
            if in_table and table_rows:
                add_table(doc, table_rows, cn_font)
                table_rows = []
                in_table = False
            i += 1
            continue

        # 表格行
        if stripped.startswith('|') and stripped.endswith('|'):
            cells = [c.strip() for c in stripped.split('|')[1:-1]]
            # 跳过分隔行 |------|------|
            if all(re.match(r'^[-:]+$', c) for c in cells):
                i += 1
                continue
            table_rows.append(cells)
            in_table = True
            i += 1
            continue

        # 非表格行时先刷新表格缓冲
        if in_table and table_rows:
            add_table(doc, table_rows, cn_font)
            table_rows = []
            in_table = False

        # H1 标题
        if stripped.startswith('# ') and not stripped.startswith('## '):
            title = stripped[2:].strip()
            p = doc.add_paragraph()
            p.alignment = WD_ALIGN_PARAGRAPH.CENTER
            p.paragraph_format.space_before = Pt(12)
            p.paragraph_format.space_after = Pt(8)
            run = p.add_run(title)
            set_run_font(run, cn_font, size=18, bold=True, color=COLORS["title"])
            i += 1
            continue

        # H2 标题
        if stripped.startswith('## ') and not stripped.startswith('### '):
            title = stripped[3:].strip()
            p = doc.add_paragraph()
            p.paragraph_format.space_before = Pt(14)
            p.paragraph_format.space_after = Pt(6)
            # 添加左侧色条效果（用粗竖线字符）
            bar = p.add_run("▎")
            set_run_font(bar, cn_font, size=14, bold=True, color=COLORS["h2"])
            run = p.add_run(title)
            set_run_font(run, cn_font, size=14, bold=True, color=COLORS["h2"])
            i += 1
            continue

        # H3 标题
        if stripped.startswith('### '):
            title = stripped[4:].strip()
            p = doc.add_paragraph()
            p.paragraph_format.space_before = Pt(10)
            p.paragraph_format.space_after = Pt(4)
            run = p.add_run(title)
            set_run_font(run, cn_font, size=12, bold=True, color=COLORS["h3"])
            # 问题下方加浅色下划线
            i += 1
            continue

        # 分隔线
        if re.match(r'^-{3,}$', stripped):
            p = doc.add_paragraph()
            p.paragraph_format.space_before = Pt(2)
            p.paragraph_format.space_after = Pt(2)
            run = p.add_run("─" * 60)
            set_run_font(run, cn_font, size=6, color=RGBColor(0xCC, 0xCC, 0xCC))
            i += 1
            continue

        # 引用块
        if stripped.startswith('> '):
            text = stripped[2:].strip()
            add_info_box(doc, text, cn_font)
            i += 1
            continue

        # 无序列表
        if stripped.startswith('- ') or stripped.startswith('* '):
            text = stripped[2:].strip()
            add_formatted_paragraph(doc, f"  \u2022  {text}", cn_font, size=10,
                                    indent=0.5)
            i += 1
            continue

        # 有序列表
        m = re.match(r'^(\d+)\.\s+(.+)', stripped)
        if m:
            num = m.group(1)
            text = m.group(2)
            # 处理链接
            text = re.sub(r'\[([^\]]+)\]\(([^)]+)\)', r'\1', text)
            add_formatted_paragraph(doc, f"  {num}.  {text}", cn_font, size=10,
                                    indent=0.5)
            i += 1
            continue

        # 复选框
        if stripped.startswith(('☐ ', '☑ ')):
            add_formatted_paragraph(doc, f"  {stripped}", cn_font, size=10,
                                    indent=0.5)
            i += 1
            continue

        # 普通段落
        add_formatted_paragraph(doc, stripped, cn_font, size=10)
        i += 1

    # 刷新最后的表格
    if in_table and table_rows:
        add_table(doc, table_rows, cn_font)

    return doc

def main():
    parser = argparse.ArgumentParser(

    doc = convert_md_to_docx(md_text, args.font)
```
