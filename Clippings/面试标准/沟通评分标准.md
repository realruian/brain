# 评估维度

1.停顿与犹豫(Pauses&Hesitation)：主要指因思考、犹豫或表达困难而产生的沉默停顿，以及伴随停顿的口语填充词(如"呢"、"嗯"等)。

2.非停顿性口头禅/习惯用语(Non-Pausing Habitual Expressions)：主要指不伴随明显停顿，但高频出现的习惯性用语(如"这个""就是说"、"然后"等)，

3.语速与语流(Pace&Flow)：语速的适中性、发音清晰度、语句衔接的连贯性，以及重复(如词语或短语的重复)对语流的影响。

4.整体听感(OverallAuditory Experience)：综合以上各项对听众理解和获取信息的影响以及听众主观的舒适度和信息接收效率

# 详细评分标准：

## 一、优秀(90-100分)

停顿与犹豫：平均每分钟出现0-1次。即使出现也极不明显，或者是非常短促(<0.5秒)的思考性停顿，不影响整体连贯性，停顿位置自然，符合语法及语义逻辑不包含呃，嗯等发声填充词

非停顿性口头禅/习惯用语：极少或几乎不出现(平均每分钟2-5次)，不影响语流，不被听众感知为累赘。

语速与语流：适中(例如中文口语约180-230字/分钟)，节奏感强，发音清晰，语流如行云流水，连贯顺畅。无冗余重复

听感：表达极其流畅，几乎没有停滞或干扰，听感舒适。语音节奏自然，句段衔接紧密，信息传递高效且专业

## 二、良好(80-89分)

停顿与犹豫：偶有轻微停顿或"呃""嗯"等发声填充词，平均每分钟1-3次，但持续时间短(<1秒)，不影响整体流畅性。偶有轻微的不自然停顿，但不影响主要句意理解。

非停顿性口头禅/习惯用语：偶尔出现(平均每分钟5-10次)。不带明显停顿，对语流影响极低，听众不会感到明显干扰。

语速与语流：适中，偶有轻微波动，但整体保持自然节奏，发音清晰。语流基本连贯。极少重复，不影响信息理解

听感：表达流畅，稍有瑕疵不影响听众理解和获取信息，听感较好。语音节奏感较好，大多数断句自然

## 三、合格(60-79分)

停顿与犹豫：存在一定数量的短促(<1.5秒)停顿或"呃""嗯"等发声填充词，平均每分钟5-8次。主要表现为思考过程中的轻微犹豫，但不干扰核心信息理解或造成语流断裂感

非停顿性口头禅/习惯用语：存在目频率可接受(平均每分钟10-15次)，但不引起明显停顿。对语流流畅性或内容理解无负面影响。听众可能注意到其存在，但不感到困扰或降低沟通效率。

语速与语流：语速适中，或有轻微波动，偶尔出现细微重复(如口头禅)，但整体信息清晰可辨，语流基本连贯，不影响重点表达，发音清晰度可能偶尔下降，但仍易于理解

听感：表达清晰可理解，信息获取无障碍。即使存在一些口头禅或思考痕迹(如停顿)整体语流保持稳定，不影响高效沟通和听感舒适度。听众无需额外努力即可接收信息

## 四、较差(30-59分)

停顿与犹豫：较多且显著，平均每分钟8-15次。部分停顿可能较长(1.5-2秒)，"嗯""呢"等发声填充词频繁出现。不自然停顿和包内断包频率高，开始影响语句完整性和信息流畅传递。

非停顿性口头禅/习惯用语：频率较高(平均每分钟15-20次)，即使不带明显停顿，其冗余性也开始一定程度地影响听感简洁性，听众可能会感到略微疲劳或分散注意力，需稍作努力过滤信息才能获取核心内容

语速与语流：语速过快导致部分含糊不清，或语速过慢且中间充斥较多停顿。断句不自然，语流开始显现轻微破碎感，重复增多并偶尔影响理解。

听感：频繁的停顿，口语化词汇和不连贯的语流开始一定程度地影响听众理解和信息获取效率。听感不佳，语流有时出现断裂感

## 五、差(0-29分)

停顿与犹豫：极其频繁地出现(平均每分钟15次以上)，或其中某类问题(如长时间停顿)极其突出。表达被长时间(2秒以上)且频繁的不自然停顿严重打断，句内断裂非常多，几乎无法形成完整语流。"呢""嗯"等发声填充词充压

非停顿性口头禅/习惯用语：极高频出现(平均每分钟20次以上)，几乎每句话都有，严重影响听感和信息接收。听众易感极度疲劳，无法有效获取信息，甚至感到言语混乱，沟通效率低下。

语速与语流：语速极不自然，可能非常快导致内容完全无法辨识，或停滞时间远多于说话时间。因大量不自然断包和重复导致语流严重中断或完全中断

听感：表达断续，难以理解，甚至几乎无法捕捉到完整信息。听感极差，语音破碎，可能导致交流无法有效进行
