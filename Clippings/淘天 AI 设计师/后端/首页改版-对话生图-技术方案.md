模型支持的尺寸

```json
gpt
    * 尺寸字符串或无选默认值为1024×1024
     * 生成图像的大小。必须是1024x1024，1536x1024（景观）之一，
     * 1024x1536（肖像）


jimeng
     * 生成图像的宽高像素，要求介于 [512 x 512, 2048 x 2048] 之间。
     * 推荐可选的宽高：
     * 1024x1024
     * 864x1152
     * 1152x864
     * 1280x720
     * 720x1280
     * 832x1248
     * 1248x832
     * 1512x648


```

### 对话页 - 提交生图（自由模式可以联调）

Url: /handler/chat/question

Method: POST application/json 

入参：

| 字段 | 说明 |  |

| generationType | 生图类型 freeMode，openScreen，productPoster，mainImageTemp | ```json<br>FREE_MODE(1,"freeMode", "自由模式"),<br>OPEN_SCREEN(2,"openScreen", "开屏图"),<br>PRODUCT_POSTER(3,"productPoster", "商品海报"),<br>MAIN_IMG_TEMP(4,"mainImageTemp", "频道版图"),<br>TAOBAO_RES_LOC(5,"taobaoResLoc", "手淘资源位"),<br>IMAGE_PROCESS(6,"imageProcess", "图像处理"),<br>PRODUCT_MAIN_IMAGE(7,"productMainImage", "商品主图"),<br>BIG_PROMOTION_REPORT(8,"bigPromotionReport", "大促战报"),<br>ANIMATION_VIDEO(9,"animationVideo", "动画视频"),<br>BATCH_EXTENSION(10, "batchExtension", "批量扩展"),<br>``` |
| editType | 编辑类型，高清hd，局部重绘partialRepaint，扩大sizeExpansion， | ```json<br>HD("hd", "高清"),<br>PARTIAL_REPAINT("partialRepaint", "局部重绘"),<br>SIZE_EXPANSION("sizeExpansion", "尺寸扩展"),<br>FILTERS("filters", "滤镜"),<br>ID_PHOTO("idPhoto", "证件照"),<br>INDUSTRIAL_CARD("industrialCard", "工牌"),<br>PET("pet", "宠物"),<br>CUTOUT("cutout", "抠图")<br>``` |
| extraParam | json ，额外的参数，比如修图参数等 | 详细字段待定<br>目前还不知道第三方接口如何调用 |
