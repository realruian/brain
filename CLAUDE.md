# Brain Vault

## Vault 结构

- `_templates/` — 笔记模板
- `00-Inbox/` — 未分类的快速笔记
- `01-Projects/` — 活跃项目的笔记和文档
- `02-Areas/` — 长期关注领域
- `03-Resources/` — 参考资料、教程
- `04-Archive/` — 已归档内容
- `05-Daily/` — 每日笔记

## 规则

- 所有笔记使用中文
- 新建笔记默认放入 `00-Inbox/`，除非明确指定位置
- 笔记使用 Markdown 格式，文件名用中文简短描述
- 每个笔记顶部加 YAML frontmatter，至少包含 tags 和 date
- 不要修改 `.obsidian/` 目录下的任何文件
- 创建或删除笔记后，不需要更新索引
