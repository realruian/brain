## 推进清单

### 结构 / 死代码

- `app/page.tsx` 移除 `ScrollProvider` 包裹

### 滚动交互修复（StickySceneTabBar）
- **吸顶"snap"**：把非吸顶的 64px 呼吸空间从 sticky 内部挪到外部独立 spacer，让 sticky 本身 padding 恒定，激活只切 position 不切尺寸
- **分割线黑一闪**：Tailwind v4 默认 border-color 是 `currentColor`（v3 是 `gray-200`）。让 `border-b` 常驻，只切颜色 `transparent ↔ black/[0.08]`，避免 `currentColor` flash
- **白底被带下来**：IntersectionObserver 异步回调 + CSS `transition-bg duration-150` + `position: sticky` 原生同步释放 → 三者叠加产生"白条拖尾"。两层修复：(1) IO → scroll + rAF，状态同帧同步；(2) 过渡非对称，stuck=true 走 150ms 平滑淡入，stuck=false 用 `duration-0` 瞬时消失

### 视觉对齐 Figma
- TopBar 透明 / 头像+"业务"保持 60px，不做"视觉直觉"式加高（踩坑：建议改到 72px 被用户按 Figma 原稿纠正）
- 吸顶白底改 `bg-white/90 backdrop-blur-md` 磨砂质感
- Tab 栏 `py-3` → `pt-6 pb-0`，下划线贴底部分割线
- TemplateCard hover 从"整图蒙层 + 居中白字 + 胶囊按钮"改成"底部 62px 黑色 `bg-black/40 backdrop-blur-[10px]` 蒙版 + 10px 白字模板名 + 194×28 橙色按钮"（按 Figma 精确规格）

### 背景视频修复
- `absolute inset-0 h-full w-full` → `fixed inset-0 h-screen w-screen`。原来 `h-full` 跟随父容器（整页文档高度几千 px）拉伸，视频被放大到视口宽数倍，只能看到中心窄条
- 基础渐变背景同步改 `fixed`
