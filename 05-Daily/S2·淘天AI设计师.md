# S2·淘天AI设计师

**产品目标：**

*   **方向：**从**基础图像能力（计件类设计）**升级为一款**全类型图像能力（全类型设计）**。
    
*   **指标：**运营使用人数提升100%，从1000人提升至2000人；覆盖75%+的设计需求；  
    \*AI覆盖的计件与驻场的设计需求，等于是覆盖了运营到团队的需求。参考计件平台1500+人。
    
*   **🌟业务侧：业务场景的能力设计覆盖与提升，从40w提升至100w以上。提升业务影响力。（提效成本至少n%。）**
    
*   **🌟能力侧：**  
    **1、（已有）完成静&动态尺寸拓展，并体验顺畅与稳定，提升基础设计需求的覆盖度到85%。**  
    **2、（新的）AI生成能力可进行整体的批量化操作，配合业务提升大量的调用。**  
    **3、（新的）体验能力优化，提升用户使用的舒畅度，生图时长不超过60s。**

    **产品功能**
    
*   小二人群：**工作类，生活类，热点类**
    
    *   **工作类：**开屏、营销会场、传播海报、大促战报、资源位、**IP公仔、商详主图、PPT图、视频动画**
        
    *   **生活类：**证件照、**宠物照、全家福、明星合影、换发型&妆照、生日照片、周年合影。**
        
    *   **热点类：24节气海报、人物盲盒、新春合影、圣诞妆容、冬季滑雪、高考、61儿童**  
        
*   设计人群：**设计类**
    
    *   **辅助设计：**抠图、扩图、批量（动态&静态）、洗图、**自由模式（创意发散）**  
        
*   综合能力：**AI agent模式，平台部署，钉钉产品部署，融合-淘天设计语义模型。**

**体系运作**

*   **能力研发：**
    
    *   **职责：持续更新AI能力，训练定制的LOAR模型，周度模型更新。**
        
    *   **3人：**1位AI全栈设计师；1位模型训练师；+1位训练图集同学\*-协作各团队需求；+1招聘正岗一位AI训练图集（也可协作业务）；校企合作项目（国美供给）  
        
*   **产品上线：**
    
    *   **职责：功能设计与开发还原，周度迭代。**
        
    *   **6人：**3位外包-前端技术（+1前端）：2位外包-后端技术（+1后端）：1位产品设计；1位-正岗AI中和技术  
          
        
*   **使用运维：**
    
    *   **职责：横向批量需求衔接；（如1400张AI海报，每周20张直播素材），每日优质案例更新。**
        
    *   **2人：**1位产品使用（运维角色，对接横向大批量需求，训练内容对接。）。1位产品更新（后台进行模型配置，案例配置，案例更新，数据）  
        

**关键问题**

*   **运营使用提升：**
    
    *   **01、已在设计团队的需求。（默认找人，AI我不用，最好你用给我图。）**  
        计件显示：1500人运营需求，一位也是核心设计对接团队。  
        适用场景：01、紧急出图，不抠细节。02、重复模版，替换内容。  
        
    *   **02、未在设计团队的需求。（没有人，AI你用一用，我试一试。）**  
        目前后台预判：横向多团队看到能力，常常过来试一下，出现AI对应需求。  
        适用场景：前台用个性化图库；
