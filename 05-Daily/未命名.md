1. Tabs 结构整改

（最大改动） - 设计稿是 8 个：全套活动 / 会场头图 / 会场组件 / 站内资源位 / 站外资源位 / 素材修改 / 图像处理 / 批量处理 - 当前是 6 个：自由模式 / 会场头图 / 会场组件 / 站内资源位 / 站外资源位 / C 端外素材 - 差异：「自由模式」→「全套活动」；「C 端外素材」删；新增「素材修改 / 图像处理 / 批量处理」

2. 创作面板从"自由输入"改成"结构化表单"

（最大功能差） - 设计稿是一组字段： - 提示句「生成一张会场头图」 - 渠道 + chip（外卖 APP） - 活动 + chip（神抢手） - 主标题 + 文字框 - 副标题 + 文字框 - 画面描述（必填） - 尺寸 宽度/高度（数字输入） - 当前是单一 textarea + 「自由模式 / 智能比例」切换 - 差异：整个面板要改版，字段结构全变

3. "AI 图像处理" section 整块删除

- 设计稿没有这节（5 个卡片不存在） - 当前有 AI延展 / AI抠图 ×3 / AI扩图 — 而且名字还重复 - 差异：这些能力可能挪到顶部「图像处理」tab 里了，首页首屏不再放

4. 模板区标题 + 布局

- 设计稿：没「全部模板」标题，直接密集网格，卡片多种比例混排（方形 / 横图 / 竖图）、带手机状态栏（20:11）和信号图标 → 呈现的是"手机上实际样子" - 当前：有「全部模板」标题 + 只 1 张 - 差异：去标题 + 网格支持多尺寸 + 模板卡片上加移动端 mockup 壳（状态栏 / 信号 / 电量）+ 模板数据量需要扩

5. Hero 区上下留白

- 设计稿：hero 文字离顶栏很近（~60px 左右） - 当前：hero 文字上方大量留白（视觉上 centered 在视口中） - 差异：需要压缩 hero 顶部 padding

6. 背景色调

- 设计稿：底部暖粉/橙渐变，整体偏暖 - 当前：整体偏冷灰 - 差异：渐变方向和颜色要调
