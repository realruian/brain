
| 步骤  | 用户操作            | 页面展示                                                    |
| --- | --------------- | ------------------------------------------------------- |
| 1   | 在首页全链路分类下选择一个模板 | 首页：展示全链路分类及模板列表![[首页.jpg]]                              |
| 2   | 点击模板进入编辑器       | 编辑器页面：默认以「会场」展示该会场的默认模板（含头图+各组件）![[Frame 123691 2.jpg]] |
| 3   | 修改会场各部分内容       | 编辑器页面：可编辑头图及各组件的文字、图片、元素![[Frame 123691 1.jpg]]         |
| 4   | 确认会场效果后选择后续操作   | 编辑器页面：提供「导出」与「一键延展」两个入口                                 |
| 5a  | 点击「导出」          | 直接导出切图至本地                                               |
| 5b  | 点击「一键延展」        | 跳转至资源位选择界面![[Frame 123695.jpg]]                         |
| 6   | 勾选需要延展的资源位      | 资源位选择界面：展示可选资源位列表![[Frame 123695 1.jpg]]                |
| 7   | 确认选择后返回编辑器      | 编辑器页面：左侧新增资源位切换区                                        |
| 8   | 切换并编辑各资源位       | 编辑器页面：根据所选资源位，分别修改对应的文字、图片、元素                           |
| 9   | 全部确认后一键导出       | 一次性导出所有资源位的切图至本地                                        |
