# DesignMind

公司内部 AI 设计图像生产平台，参考淘天 AI 设计师内部系统独立搭建。

代码仓库：`~/Desktop/Dev/designmind`

---

## 项目背景

公司内部设计师和运营人员需要高频生产电商设计素材（海报、版头、主图等），现有流程依赖人工，耗时长、成本高。

参考淘天内部系统（已有完整的 API 文档、UI 交互稿、技术方案文档），独立实现一套类似能力的平台。参考文档存放在 `designmind/doc/`，共 22 份（产品 PRD、后端接口、前端规范、UI 截图）。

---

## 项目目标

- **核心**：对话输入 → AI 生图（SSE 流式），支持文生图、图生图、多轮修改
- **延展**：模版选择 + 一键多尺寸生图延展
- **工具**：AI 扩图、AI 抠图、AI 证件照等独立图像处理工具
- **记录**：历史对话和生成图片可回溯

---

## 技术选型理由

| 技术                  | 选择理由                            |
| ------------------- | ------------------------------- |
| Next.js 14 全栈       | SSE 原生支持，全栈单仓库，内部工具无需拆微服务       |
| Vercel AI SDK       | 统一抽象多模型（OpenAI / 火山引擎），SSE 开箱即用 |
| PostgreSQL + Prisma | 关系型存储满足需求，Prisma 类型安全，迁移管理简单    |
| Zustand             | 轻量状态管理，适合聊天会话状态                 |
| GPT-Image-1 为主模型    | 当前最高质量图像生成，支持 edit 接口（扩图/修图）    |
| shadcn/ui           | 快速还原设计稿，无样式绑定，可完全自定义            |

---

## MVP 范围

**包含（Phase 1-5，约 10 天）：**
- 首页：模型选择器 + 输入框 + 模版网格
- 对话生图：SSE 流式，骨架占位 → 真实图片，4 宫格展示
- 历史记录：左侧侧栏，按今天 / 7 天内 / 30 天内分组
- 中断确认：图片意图澄清、生成计划确认
- 图像工具：AI 扩图（含常用尺寸面板）、AI 抠图

**暂不包含：**
- 批量生产（表格导入）
- 提示词 RAG 自动补全
- 管理后台（模版 / 配额管理）
- 模型训练 / LoRA 定制
