## 一、概述

本标准基于美团设计部在品牌文创领域的长期实践积累，结合行业成熟的设计评估方法论，在日常项目评审与复盘中提炼形成，作为设计部品牌文创方向的统一质量评估依据。

品牌文创设计的核心命题，是让品牌价值从抽象的视觉符号走向用户可感知、可触摸、可传播的具体形态。无论是一件文创礼品、一套品牌标志，还是一个 IP 角色，最终都要回答同一个问题：它是否真正为品牌、为用户、为业务创造了可被验证的价值？基于这一出发点，本标准将"价值"作为三类项目共同的核心评估轴，同时针对各类型的设计本质，分别构建差异化的专业维度体系。

- **文创礼品：**从创意感、实用性、传播力三个维度构建专业评估体系。文创礼品连接的是品牌与用户之间最具温度的一次触达——它既要承载品牌文化主张，又要融入用户的真实生活，还要具备让人主动分享的内在驱动力。这四个维度正是对"一件好的文创礼品应该是什么样的"这一问题的系统性回答：有故事、有用、有晒点，并且能说清楚它带来了什么改变。本标准为文创礼品的设计决策提供了清晰的专业评估依据，推动品牌文创从"好看的礼物"走向"有价值的品牌资产"。

- **品牌标志 Logo/VI**：基于保罗·兰德 Logo 设计七原则，结合设计行业专业实践与美团设计部工作积累，选择简洁性、可视性、独特性、可记忆性、适应性、普适性、永恒性七个维度构建评估体系。品牌标志是企业最持久的视觉资产，它需要在任何尺寸、任何媒介、任何文化语境下都能被清晰识别，并在数年乃至数十年的时间跨度中保持生命力。这七个维度正是对"一个好的品牌标志应该经得起什么考验"的系统性回答。本标准有助于在团队内部及协同方之间建立统一的评审语言，推动品牌设计从"好看"走向"有效"、从主观感受走向可量化评估。

- **IP 角色：**选择从秒懂、秒记、秒传播、价值四个维度构建评估体系。IP 角色是品牌情感化表达的重要载体，在美团多业务场景中承担着强化品牌与用户情感连接的关键职能。一个好的 IP，不仅要让用户一眼看懂它代表谁、记得住它长什么样，更要具备让用户主动传播、自发二创的内在驱动力。"秒懂、秒记、秒传播"三维框架，正是对 IP 设计从品牌认知到传播势能全链路的系统性评估，推动品牌 IP 设计从"好看"走向"有传播力"、从情感表达走向可量化的品牌资产建设。

## 二、标准评估细则

#### 1.文创礼品

| **创意感**<br>品牌识别清晰｜文化叙事完整｜设计创意独特 | **实用性**<br>功能实用有价值｜使用体验愉悦｜材质工艺考究 | **传播力**<br>有晒点｜有话题性｜引发自传播 |
| --- | --- | --- |
| **实际价值**<br>业务价值｜经验价值｜社会价值（[实际价值评估标准](https://km.sankuai.com/collabpage/2760784088)） |  |  |

### 创意感（权重 35%）

**品牌性、文化内涵、创意性**

文创产品是否承载并传递了品牌的文化主张与价值观，能引起情感共鸣。有独特的创意巧思或视觉亮点在同类产品中具有差异化辨识度。

**90 评估标准**

1. 品牌识别清晰，产品本身就是品牌的有形载体，可清晰传递品牌是谁、代表什么。

2. 文化叙事深度与品牌高度吻合，文化主张清晰独特，设计有立场、有温度，成为品牌文化资产的有形载体 。

3. 拥有独特的创意巧思，具有较强的差异化辨识度；视觉元素与细节都经过精心推敲，视觉品质高。

**80 评估标准**

1. 用户能感知产品与品牌的关联，有一定的情感共鸣 。

2. 文化故事清晰可感知，设计决策有明确的文化依据，而非纯粹装饰 。

3. 有一定的视觉记忆点，在同类产品中具有可感知的差异化；视觉元素细节精致合理，视觉品质较高。

### 实用性（权重 35%）

**好用、有真实价值**

产品是否具有真实的使用价值，能融入用户的日常生活，使用体验感佳。材质选择考究，工艺精良。

**90 评估标准**

1. 功能实用，设计有亮点，解决了用户真实痛点或情感诉求，用户愿意购买和使用。

2. 充分体现"为美好生活而设计"的核心理念，使用过程流畅且令人愉悦，有一定的可持续性考量，体现对社会责任的关注 。

3. 材质选择考究，工艺精良，触感舒适，耐用性优异，材质与工艺本身成为产品体验的加分项。

**80 评估标准**

1. 提供真实的功能或情感价值，用户愿意在日常生活中使用 。

2. 使用体验顺畅，基本解决了用户的实际需求，无明显体验障碍 。

3. 材质工艺合理，触感与耐用性达标，品质感与价格预期匹配。

### 传播力（权重 30%）

**有话题、有晒点**

产品是否具有让用户主动分享、自发传播的内在驱动力。是否有让人眼前一亮的创意或设计细节，能引发社交媒体上的自发传播和讨论。

**90 评估标准**

1. 产品具有"晒的价值"，拍照好看、有故事可讲、有话题可聊，用户收到后会主动分享推荐 。

2. 设计本身就是一个值得被传播的内容，有让人眼前一亮的创意或设计细节，有话题性和讨论价值，能引发自传播 。

3. 品牌与产品深度融合，具有收藏价值或限定属性，传播的同时强化了品牌认知，能在社交媒体上引发规模化的自发传播和讨论。

**80 评估标准**

1. 产品有一定的"晒点"，用户有分享意愿。

2. 有一定的设计亮点或话题性，收到或使用时会主动跟人提起 。

3. 有一定的收藏价值或限定属性，在目标受众中具备传播潜力。

| **适应性** | **延展性、跨媒介**<br>品牌核心的视觉符号标志在各种应用中都有良好的视觉表现，在不同媒介上都能保持其识别性和视觉美感。 | 15% | 无需大幅修改就能在各种媒介和尺寸下保持视觉一致性和吸引力。 | 可在多种常规场景应用，但需进行调整，部分场景下可能失去原有设计美感。 |
| **普适性** | **无歧义、无禁忌**<br>在被目标消费者解读的时候不会产生误解或反触的情绪，并且在传播上尽可能使所有目标消费者理解一致。 | 10% | 跨越文化、年龄、性别等因素，被广泛受众理解和接受，无负面联想。 | 大部分主流群体可接受，但在特定小众群体或特殊场景下可能有问题。 |
| **永恒性** | **前瞻性、经典不过时**<br>品牌形象识别通常会被企业沿用五年甚至更久，所以在设计上必须提供一个理性而富有前瞻性、考虑全面的系统架构。设计时不能追逐短期的潮流，而要考虑到标志在未来很长一段时间内都能保持其有效性和吸引力。 | 5% | 具有 timeless 的特质，10年以上都不会显得陈旧，能适应行业和审美趋势演变。 | 能适应未来 3-5 年的市场变化，但长期看可能需要频繁更新。 |

### 秒懂（权重 35%）

**品牌识别、角色定位、广泛认知**

IP的形象身份、行业属性与品牌归属容易被用户看懂，认知成本低，无需额外解释。

**90 评估标准**

1. 品牌叙事完整，视觉符号与品牌调性高度关联，IP形象本身就是品牌的有形载体。

2. IP的性格、职业或身份特征鲜明立体，用户一眼即可理解其定位，角色本身具有感染力。IP背后有完整且深度契合品牌战略的品牌故事，设计表达清晰直白兼具温度，IP形象成为品牌文化的有形载体。

3. 目标用户能自然理解IP的品牌含义，零认知门槛，无需任何额外解释即可完成从形象到品牌的完整联想。

**80 评估标准**

1. 品牌叙事较完整，视觉符号与品牌调性关联紧密。

2. 角色定位清晰易懂，用户能快速理解IP的性格或身份特征 。IP背后有明确的品牌故事或设计概念，设计表达直白无歧义。

3. 用户认知成本较低，目标受众能在较短时间内理解IP与品牌的关联。

### 秒记（权重 35%）

**辨识度、独特性、丰富性**

IP具有明确记忆点，结构简洁，特征突出，让用户看过后能记得住、说得出，整体设计语言自成体系，与同类IP有效区隔。

**90 评估标准**

1. 拥有独特的外形特征（颜色/比例/标志性元素），记忆点稀缺独特，具备过目不忘的视觉辨识度 。

2. 表情系统丰富且层次分明，动作语言生动自然，情感表达力强，具备角色生命力 。

3. 整体设计语言简洁高级、自成体系，与同类IP有鲜明区隔，形成不可复制的视觉风格。

**80 评估标准**

1. 外形特征清晰易辨，记忆点明确突出，多次接触后能记住并说得出 。

2. 表情系统基本完整，动作语言自然，情感表达清晰可感知 。

3. 设计简洁规范，整体风格统一，与同类品牌的IP形象可有效区隔。

### 秒传播（权重 30%）

**传播力、延展性、二创友好**

IP自带社交属性，能被简单描述和传递，能轻易的讲给别人听明白。适合多形式延展、二创、社媒晒图等话题讨论，能驱动用户自发传播。

**90 评估标准**

1. 形象具备鲜明特质，能被简单描述和传递，能轻易的讲给别人听明白。自带强社交传播基因，用户自发晒图分享欲强 。

2. 易于表情包、插画动画、文创周边等多元设计延伸，延展形态丰富且品质稳定。

3. 用户愿意主动二创、玩梗，具备强UGC生产的内在驱动力，形象开放度高 。话题性与讨论度突出，具备长期破圈与持续传播的潜力，能在目标圈层之外引发关注。

**80 评估标准**

1. 形象特质较鲜明，能被描述和传递，花一定时间能讲给别人听明白。自带一定社交属性，有基础传播吸引力 。

2. 能进行表情包、文创周边等基本设计延伸，延展效果协调可用 。

3. 通过设计延伸能够让用户愿意主动二创、玩梗与传播，有一定的UGC潜力 。在目标受众中有一定话题性，具备传播基础，有向更广圈层扩展的可能性。
